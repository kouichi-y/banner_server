# BannerServer

これは文字列を大きな文字で返してくれるWebアプリケーションです。

## 使用例

```bash
curl http://127.0.0.1:5000/HelloWorld!
```

```text
 _   _      _ _    __        __         _     _ _ 
| | | | ___| | | __\ \      / /__  _ __| | __| | |
| |_| |/ _ \ | |/ _ \ \ /\ / / _ \| '__| |/ _` | |
|  _  |  __/ | | (_) \ V  V / (_) | |  | | (_| |_|
|_| |_|\___|_|_|\___/ \_/\_/ \___/|_|  |_|\__,_(_)
```

banner文字列であれば、カタカナもいけます。

```bash
curl http://127.0.0.1:5000/banner/コンニチワ
```
